1. JK Flip-Flop – Truth Table & Working

A JK Flip-Flop is the most versatile flip-flop. It eliminates the invalid state of the SR flip-flop. It has two inputs J (Set) and K (Reset), along with a Clock (CLK).

Truth Table

JKCLKQ (next)Q̄ (next)Operation
00↑QQ̄No Change
01↑01RESET
10↑10SET
11↑Q̄QToggle (complement)
💡 Key difference from SR: When J=1, K=1 the output toggles (flips) instead of going into an invalid state.

Characteristic Equation

Q(next) = J·Q̄ + K̄·Q

Logic Diagram of JK Flip-Flop

Working Explanation

  • J=0, K=0: Both AND gates output 0 → SR = 00 → No change (Memory state)
  • J=0, K=1: With Q=1 feedback → R=1, S=0 → Output resets to 0
  • J=1, K=0: With Q̄=1 feedback → S=1, R=0 → Output sets to 1
  • J=1, K=1: Output complements/toggles on each clock pulse

8. Advantages of Multiplexer

  • Reduces number of data transmission lines (N-to-1)
  • Saves hardware cost and board space
  • Can implement any Boolean function directly
  • Increases data bandwidth and reduces circuit complexity
  • Can be used as a universal combinational circuit
  • Easy to cascade for larger MUX (e.g., two 4:1 → 8:1)

9. 4:1 MUX using 2:1 MUX

Three 2:1 MUXes are needed to build one 4:1 MUX.

Level 1: Two 2:1 MUXes select among I₀,I₁ (controlled by S₀) and I₂,I₃ (controlled by S₀). Level 2: One 2:1 MUX selects between outputs of Level 1 (controlled by S₁).

10. Combinational Circuit

A combinational circuit is a digital circuit whose outputs depend only on present inputs, with NO memory or feedback.

Characteristics

  • Output = f(current inputs only)
  • No clock signal needed
  • No memory element
  • Described by Boolean equations

Examples

  • Half Adder / Full Adder
  • Multiplexer / Demultiplexer
  • Encoder / Decoder
  • Comparator

13. Race Around Condition in JK Flip-Flop

When J=1 and K=1, the output should toggle. But if the clock pulse duration is long, the output may toggle multiple times within a single clock period — this is called the Race Around Condition.

Why it Occurs

The propagation delay through the flip-flop is less than the clock pulse width. So Q̄ feeds back to the input, causing repeated toggling during the HIGH period of the clock.

⚠️ Race occurs when: Clock pulse width > propagation delay of flip-flop

Solutions

SolutionDescription
Master-Slave JK FFTwo latches in series; output changes only on clock edge, not during pulse
Edge-Triggered FFState changes only on rising or falling edge of clock (not during high/low period)
Reduce clock pulse widthMake pulse width less than propagation delay (difficult in practice)
✅ Best solution: Use Edge-Triggered JK Flip-Flop or Master-Slave configuration.

17. Steps for Combinational Circuit Design

  1. Step 1: Understand the problem and identify number of inputs and outputs
  2. Step 2: Assign binary variables to inputs and outputs
  3. Step 3: Construct the truth table for all input combinations
  4. Step 4: Obtain simplified Boolean expression using K-map or algebraic simplification
  5. Step 5: Implement the simplified expression using logic gates
  6. Step 6: Verify the circuit against the truth table

18. Full Adder using Two Half Adders

A Full Adder can be built using two Half Adders (HA) and one OR gate.

Stage 1 (HA1): S₁ = A ⊕ B, C₁ = A·B
Stage 2 (HA2): Sum = S₁ ⊕ Cin, C₂ = S₁·Cin
Final: Cout = C₁ + C₂

21. Sequential Circuit – Definition & Example

A sequential circuit is a digital circuit whose output depends on both present inputs AND past history (stored state). It contains memory elements (flip-flops).

Mealy Machine

Output depends on present input AND present state

Output = f(State, Input)

Moore Machine

Output depends only on present state (not on input directly)

Output = f(State)

Examples

  • Flip-flops (SR, JK, D, T)
  • Counters (up, down, MOD-N)
  • Shift Registers
  • Sequence detectors
  • Traffic light controller
